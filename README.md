# Short-Reach Optical Interconnect Research

This repository stores version-controlled research reports and maintenance rules for short-reach optical interconnects, SerDes, optical modules, AI server fabrics, and AI data center infrastructure.

## Scope

Focus areas:

- Co-packaged optics (CPO), silicon photonics, near-packaged optics (NPO), optical engines, VCSEL/EML/CWL paths.
- SerDes and high-speed electrical I/O: 224G/448G PAM4, ADC/DSP-based transceivers, Retimer/Redriver, PCIe/CXL/UALink/ESUN/UEC links.
- Optical module industry: 800G, 1.6T, 3.2T, LPO, OSFP, QSFP-DD, silicon photonics modules.
- AI server and super-node infrastructure: NVIDIA NVL systems, UALink, ESUN, scale-up and scale-out fabrics.
- Compute facility infrastructure: liquid cooling, immersion cooling, HVDC, rack power density, distributed training network architecture.

Out of scope:

- Consumer electronics, mobile SoCs, automotive chips, IoT devices, and bitcoin mining.
