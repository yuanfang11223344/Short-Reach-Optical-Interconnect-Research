## 🔌 SerDes与互连技术

1. 🔥 **IEEE P802.3dj Draft 3.1 评论处理成为 200G/lane 主线**
   - Draft 3.1 received comments 已公开，覆盖 electrical/optical PMD、FEC 与管理条款；下一阶段重点是 ballot comment 收敛，而非继续扩张 baseline。
   - 技术视角：224G SerDes 的验证边界已从单 IP 指标扩展到 package、PCB、connector、retimer 与 FEC 的系统闭环。
   - 来源：https://www.ieee802.org/3/dj/comments/index.html

2. ⭐ **OIF CEI-224G-LR/MR 与 IEEE 802.3dj 保持 liaison 对齐**
   - IEEE 五月材料记录了 CEI-224G-LR-PAM4 与 CEI-224G-MR-PAM4 项目进展，说明长通道和中距离 channel budget 仍在持续收敛。
   - 来源：https://ieee802.org/3/dj/public/26_05/index.html

3. ⭐ **Credo 224G multiprotocol retimer 面向 UALink、ESUN 与 Ethernet**
   - Blue Heron 将多种 scale-up/scale-out fabric 放在同一 224G PHY 平台，协议复用开始成为 retimer 的核心产品属性。
   - 来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Industrys-First-224G-Multiprotocol-AI-Scale-Up-Retimer-Supporting-UALink-ESUN-and-Ethernet/default.aspx

## 🔦 光学与光模块

1. 🔥 **OpenLight 3.2T DR8 PIC 将单通道推进到 400G/lane**
   - 官方宣布 3.2T DR8 silicon photonics PIC 样片及 1.6T DR8 LRO/LPO variants，路线采用异质集成 laser 与 EAM。
   - 技术视角：真正的门槛是 400G/lane E/O、driver/TIA、封装损耗和量产测试共同成熟。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/openlight-introduces-3-2t-dr8-silicon-photonics-pic-s-as-well-as-1-6t-dr8-lro-and-lpo-variants/

2. ⭐ **Coherent 的 6.4T socketed CPO 强调可维护性折中**
   - OFC 2026 演示覆盖 32×200G socketed CPO、external laser source、VCSEL CPO 与 400G modulator。
   - 来源：https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026

3. ⭐ **Lightmatter L20 同时支持 NPO 与 OBO 集成**
   - Passage L20 以 6.4Tbps/方向的 BiDi optical engine 覆盖多机架 scale-up 与 scale-out，反映 CPO 量产模型定型前多形态并存。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/lightmatter-expands-photonic-interconnect-roadmap-with-passage-l20-unified-optical-engine/

4. 📌 **国内光模块制造链开始为 3.2T 工艺预留能力**
   - NEPCON ASIA 2026 的光模块示范线覆盖 800G/1.6T/3.2T 的固晶、键合、耦合、SMT、点胶和测试老化环节。
   - 来源：https://www.nepconasia.com/zh-cn/news-center/zhxx/2026/6/1.html
