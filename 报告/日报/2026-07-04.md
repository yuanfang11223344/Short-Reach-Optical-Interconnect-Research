# 芯片互连每日资讯 2026-07-04

> 检索窗口：截至 2026-07-04（Asia/Shanghai）。周末一手新增有限，本期采用“近期新增优先 + 官方资料复核”，不把历史公告包装为当天发布。

## 🔌 SerDes与互连技术

1. 🔥 **IEEE P802.3dj Draft 3.1 进入 SA ballot comment 收敛。** 6月22日公开的 received comments 覆盖 200G/lane electrical/optical PMD、FEC 与管理条款，量产验证已转向 package、channel、DSP 和测试模型的一致性。[IEEE](https://www.ieee802.org/3/dj/comments/index.html)
2. 🔥 **OIF 同时推进 CEI-448G-VSR 与 CEI-448G-LR。** 官方 Current Work 已将 chip-to-module 和 backplane 两类 448G interface 列为项目，意味着 448G 从 framework 进入 reach 与 channel budget 定义。[OIF](https://www.oiforum.com/technical-work/current-work/)
3. ⭐ **Broadcom Taurus 将 optical DSP 推到 400G/lane。** 官方定位采用 8:4 架构服务 1.6T 模块，并为 3.2T optical module 的 400G/lane 路线铺路。[Broadcom](https://investors.broadcom.com/news-releases/news-release-details/broadcom-delivers-industrys-first-400glane-optical-dsp-next)
4. ⭐ **UEC 1.0.2 仍是当前 Ultra Ethernet 实现基线。** 版本修正 CMS congestion-control algorithms；scale-out 竞争正从 PHY 速率上移到 transport、拥塞控制和多厂商互操作。[UEC](https://ultraethernet.org/specification-history/)

## 🔦 光学与光模块

1. 🔥 **1.6T 商用与 3.2T/400G-lane 预研形成代际重叠。** 7月4日产业报道集中复盘 OFC 2026 的 XPO/CPX、NPO/CPO 与 400G/lane 路线；其中产品指标仍需回到厂商一手材料核验。[报道](https://inf.news/en/tech/4e0b3152e4596ad628b8d84734d2d6be.html)
2. 🔥 **Coherent 6.4T socketed CPO 覆盖 SiPh、VCSEL 与 InP 多路线。** 32×200G optical engine 配合 ELS，体现 CPO 的竞争焦点已包含 serviceability、fiber attach 和 advanced packaging。[Coherent](https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026)
3. ⭐ **NLM Photonics 的 1.6T/3.2T SOH PIC 进入 sampling。** SOH 调制路线提供低驱动电压和高带宽潜力，但真正产品化仍取决于材料稳定性、foundry process control 与长期可靠性。[OFC](https://www.ofcconference.org/news-media/exhibitor-news/nlm-photonics-initiates-sampling-of-1-6t-and-3-2t-silicon-organic-hybrid-pics/)
4. ⭐ **国内 GIGALIGHT 发布 1.6T DR16 NPO linear SiPh engine。** 官方强调与传统 DR4 模块混合互通；当前应视为公开产品节点，不等同于规模客户部署。[GIGALIGHT](https://www.gigalight.com/news-events/news-10133.html)

## 🖥️ AI服务器与超节点

1. 🔥 **UALink 2.0 把 In-Network Compute 引入开放 scale-up fabric。** reduction、aggregation 和 synchronization 可在 fabric 中执行，使 switch ASIC 从纯转发器上移为 collective acceleration 节点。[UALink](https://ualinkconsortium.org/blog/exploring-in-network-compute-how-ualink-is-redefining-ai-scale-up-architecture-1509/)
2. 🔥 **Vera Rubin 以六芯片 extreme codesign 定义 rack-scale 系统。** Vera CPU、Rubin GPU、NVLink 6、ConnectX-9、BlueField-4 与 Spectrum-6 共同决定训练、推理、存储和网络效率。[NVIDIA](https://nvidianews.nvidia.com/news/rubin-platform-ai-supercomputer)
3. ⭐ **Spectrum-X Ethernet Photonics 将 CPO 与 Rubin 平台共同验证。** 官方把 CPO switch systems 的功耗和 uptime 优势放入系统级叙事，说明 CPO 首批量产更依赖封闭平台闭环。[NVIDIA](https://nvidianews.nvidia.com/news/rubin-platform-ai-supercomputer)

## 🏢 计算中心与基础设施

1. 🔥 **AI data center 供电演进从 48V 延伸到 facility-level DC 和中压变换。** 论文将高变比 DC/DC、LVDC 与 medium-voltage solid-state transformer 分成三类关键 building blocks。[arXiv:2606.25095](https://arxiv.org/abs/2606.25095)
2. ⭐ **液冷供应链从冷板扩展到 CDU 与 chiller 系统。** Panasonic 已在欧洲提供 400/800kW CDU 和更高容量 free-cooling chiller，说明设施侧液冷接口正在标准化和模块化。[Panasonic](https://news.panasonic.com/global/press/en260304-2)
3. ⭐ **电网与 data center 可持续性进入政策和系统研究层。** 欧盟项目把 AI for grids 与 data-centre sustainability 放入同一研究框架，显示负载调度和能源系统将更紧密耦合。[European Commission](https://energy.ec.europa.eu/news/flagship-projects-ai-grids-and-data-centre-sustainability-2026-06-04_en)
