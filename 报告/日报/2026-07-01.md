# 芯片互连每日资讯 2026-07-01

> 生成时间：2026-07-01（Asia/Shanghai）。本期按近一周新增优先整理；对标准和产品路线的持续状态均明确标注，不将历史公告包装为当日发布。

## 🔌 SerDes与互连技术

1. 🔥 **OIF CEI-448G-VSR/LR 已进入正式项目阶段**
   - OIF Current Work 已列出 CEI-448G-VSR chip-to-module 与 CEI-448G-LR backplane 项目，448G electrical I/O 从 framework 进入具体 reach 定义。
   - 技术视角：相比 224G PAM4，448G 的 ADC/DAC 带宽、clock jitter、DSP equalization、package loss 与 compliance correlation 需要同步升级。
   - 来源：https://www.oiforum.com/technical-work/current-work/

2. 🔥 **IEEE P802.3dj Draft 3.1 继续处理 200G/lane SA ballot comments**
   - 6月22日公开的 received comments 覆盖 electrical/optical PMD、FEC 与管理条款，当前重点是规范收敛和跨层一致性。
   - 技术视角：224G SerDes 的量产风险已从单 IP 指标扩展到 package、PCB、connector、retimer、FEC 和 test fixture 的完整链路。
   - 来源：https://www.ieee802.org/3/dj/comments/index.html

3. ⭐ **224G multiprotocol retimer 开始承载 UALink、ESUN 与 Ethernet 共平台需求**
   - Credo Blue Heron 的多协议定位说明同一 PHY 平台需要兼顾 latency、BER telemetry、lane repair、link training 与不同 fabric 的 compliance。
   - 来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Industrys-First-224G-Multiprotocol-AI-Scale-Up-Retimer-Supporting-UALink-ESUN-and-Ethernet/default.aspx

4. ⭐ **UEC 1.0.2 仍是当前 Ultra Ethernet 实现基线**
   - UEC specification history 将 1.0.2 标为 current published version，修正 CMS congestion-control algorithms 并加入编辑性澄清。
   - 技术视角：Scale-out Ethernet 的差异化正从 PHY 速率上移到 congestion control、transport semantics 和 multi-vendor interoperability。
   - 来源：https://ultraethernet.org/specification-history/

## 🔦 光学与光模块

1. 🔥 **NLM Photonics 推进 1.6T/3.2T silicon-organic hybrid PIC sampling**
   - 其 SOH PIC 面向 1.6T 与 3.2T optical interconnect，代表除传统 SiPh+InP EAM 之外的高速低驱动电压调制路线。
   - 技术视角：产业价值取决于材料稳定性、wafer-scale process control、封装兼容性和长期可靠性，而不只是实验室带宽。
   - 来源：https://www.optica.org/about/newsroom/corporate_member_news/2026/nlm_photonics_initiates_sampling_of_1_6t_and_3_2t_silicon_organic_hybrid_pics/

2. 🔥 **OpenLight 的 3.2T 路线进入 PIC 样片与量产订单并行阶段**
   - OpenLight 官方资料显示 3.2T DR8 silicon photonics PIC 已 sampling，同时 800G/1.6T laser-integrated PIC 获得首批 volume production orders。
   - 来源：https://openlightphotonics.com/newsroom/openlight-enters-3-2t-era-with-first-volume-production-orders

3. ⭐ **Coherent 6.4T socketed CPO 强调可维护性与多材料平台**
   - OFC 2026 演示包括 32×200G socketed CPO、external laser source、VCSEL CPO 和 400G InP-on-silicon modulator。
   - 技术视角：socketed optical engine 是 fully integrated CPO 与现场可维护性之间的重要工程折中。
   - 来源：https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026

4. ⭐ **国内头部光模块厂商确认 1.6T 已量产，3.2T 仍在研发完善**
   - 中际旭创公开投资者记录称 1.6T 已批量出货并持续起量，3.2T 尚处于研发完善阶段，NPO 正推进技术完善和客户验证。
   - 来源：https://static.cninfo.com.cn/finalpage/2026-05-15/1225311210.PDF

## 🖥️ AI服务器与超节点

1. 🔥 **UALink 2.0 将 In-Network Compute 纳入开放 scale-up fabric**
   - UALink 官方技术解读指出，INC 允许 collective operations 在数据传输过程中执行，用于降低 accelerator 间同步和数据移动开销。
   - 技术视角：scale-up switch 从纯转发器件上移为计算协同节点，对 switch pipeline、reduction precision、fault handling 和 software collective library 提出新要求。
   - 来源：https://ualinkconsortium.org/blog/exploring-in-network-compute-how-ualink-is-redefining-ai-scale-up-architecture-1509/

2. ⭐ **UALink 2.0 同时扩展 chiplet、manageability 与 200G performance 定义**
   - 官方公告将 In-Network Compute、Chiplet Definition、Manageability 和 200G DL/PL 作为一组规范发布，目标是支持多 workload 的开放 accelerator fabric。
   - 来源：https://ualinkconsortium.org/wp-content/uploads/2026/04/UALink-2.0-Specification-PR_FINAL.pdf

3. 🔥 **Vera Rubin NVL72 继续强化 scale-up 与 scale-out 共设计**
   - NVIDIA 官方平台由 72 Rubin GPU、36 Vera CPU、NVLink 6、ConnectX-9 与 BlueField-4 组成，并通过 Quantum-X800/Spectrum-X 扩展集群。
   - 技术视角：节点性能兑现依赖 coherent compute domain、NVLink scale-up、Ethernet/InfiniBand scale-out、storage 和液冷共同闭环。
   - 来源：https://nvidianews.nvidia.com/news/nvidia-vera-rubin-platform

## 🏢 计算中心与基础设施

1. 🔥 **AI 数据中心供电研究从 48V 延伸到 facility-level DC 与中压变换**
   - arXiv:2606.25095 系统梳理高变比 DC/DC、LVDC distribution 与 medium-voltage solid-state transformer 路线。
   - 技术视角：rack power density 和负载瞬态继续上升时，供电拓扑会直接影响 GPU 可用功率、铜损、故障域和扩容速度。
   - 来源：https://arxiv.org/abs/2606.25095

2. 🔥 **FERC 将 large-load 并网规则提升为 AI 基础设施关键路径**
   - 六个区域电网运营方被要求解释或改革 data center 等大负载接入规则，供电可得性正影响 cluster 上线周期。
   - 来源：https://www.ferc.gov/news-events/news/ferc-launches-aggressive-targeted-action-speed-large-load-integration

3. ⭐ **800VDC、液冷与 microgrid 开始作为统一系统设计**
   - Delta 的公开方案把 800VDC architecture、power、cooling 与 microgrid 组合，反映 rack 与 facility 的接口正在前移。
   - 来源：https://www.deltaww.com/en-US/press/40334
