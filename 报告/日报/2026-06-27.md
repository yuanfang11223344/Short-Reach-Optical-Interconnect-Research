# 芯片互连每日资讯 2026-06-27

> 生成时间：2026-06-27（Asia/Shanghai）。今日公开新增有限；以下按“近一周新增优先、关键标准与产品状态复核”整理，来源以标准组织、论文和厂商官方资料为主。

## 🔌 SerDes与互连技术

1. 🔥 **IEEE P802.3dj Draft 3.1 已进入 SA ballot 评论处理阶段**
   - 要点：工作组在 6 月 22 日公开 Draft 3.1 的 received comments，覆盖 200G/lane electrical/optical PMD、FEC 与管理条款。
   - 技术视角：1.6TbE 从 baseline 选择进入规范收敛期，SerDes、DSP、module 与测试团队需要围绕 Draft 3.1 跟踪一致性风险。
   - 来源：https://www.ieee802.org/3/dj/comments/index.html

2. ⭐ **OIF CEI-224G-LR/MR 与 802.3dj 的接口边界继续对齐**
   - 要点：IEEE 802.3dj 五月会议公开了 OIF 关于 CEI-224G-LR-PAM4、CEI-224G-MR-PAM4 的进度 liaison。
   - 技术视角：200G/lane 系统能否量产，不只取决于 PHY IP，还取决于 package、PCB、connector 与 retimer 的 channel budget 是否可闭合。
   - 来源：https://ieee802.org/3/dj/public/26_05/index.html

3. ⭐ **224G multiprotocol retimer 开始面向 UALink、ESUN 与 Ethernet 共平台**
   - 要点：Credo Blue Heron 将 224G retimer 面向多种 AI scale-up/scale-out 协议，反映同一物理层复用多个 fabric 的产品趋势。
   - 技术视角：对 SerDes 设计而言，协议复用会把 compliance、latency、BER telemetry 与 lane repair 推到同一器件中权衡。
   - 来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Industrys-First-224G-Multiprotocol-AI-Scale-Up-Retimer-Supporting-UALink-ESUN-and-Ethernet/default.aspx

## 🔦 光学与光模块

1. 🔥 **OpenLight 3.2T DR8 PIC 把单通道路线推进到 400G/lane**
   - 要点：OpenLight 宣布 3.2T DR8 silicon photonics PIC 样片，并同步提供 1.6T DR8 LRO/LPO variants；其官方材料明确采用异质集成 laser 与 EAM 路线。
   - 技术视角：3.2T 的关键不只是 aggregate bandwidth，而是 400G/lane 调制器、driver/TIA、封装损耗与可测性共同成熟。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/openlight-introduces-3-2t-dr8-silicon-photonics-pic-s-as-well-as-1-6t-dr8-lro-and-lpo-variants/

2. ⭐ **Coherent 展示 6.4T socketed CPO 与 400G InP-on-silicon modulator**
   - 要点：官方 OFC 2026 演示包括 32×200G socketed CPO、external laser source、VCSEL CPO 和 400G modulator。
   - 技术视角：socketed optical engine 试图在 CPO 功耗密度与现场可维护性之间取得工程折中。
   - 来源：https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026

3. ⭐ **Lightmatter L20 以 6.4Tbps/方向覆盖 NPO 与 OBO**
   - 要点：Passage L20 使用 BiDi 架构，定位 multi-rack scale-up 与 scale-out，并允许客户选择 NPO 或 OBO 集成形态。
   - 技术视角：统一 optical engine 说明产业仍在等待 CPO 的制造与维护模型定型，NPO/OBO 会继续承担过渡和分层部署角色。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/lightmatter-expands-photonic-interconnect-roadmap-with-passage-l20-unified-optical-engine/

4. 📌 **国内 HYC 在 OFC 展示 MCF 与 CPO passive interconnect 组合**
   - 要点：方案覆盖 MCF fan-in/fan-out、FAU、patch cord 与高纤芯密度连接。
   - 技术视角：CPO 扩展会把瓶颈从光引擎延伸到 fiber attach、connector density、维修与良率追踪。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/hyc-showcases-multi-core-fiber-and-cpo-interconnect-solutions-at-ofc-2026/

## 🖥️ AI服务器与超节点

1. 🔥 **Supermicro 发布 Vera Rubin NVL4 的 HPC/AI DCBBS blueprint**
   - 要点：6 月 22 日官方公告把 compute、networking、advanced liquid cooling、power distribution 与 site infrastructure 纳入同一交付蓝图。
   - 技术视角：AI server 的竞争单元继续从主板/节点上移到 rack 与 facility，互连和冷却必须在系统验证阶段共同闭环。
   - 来源：https://ir.supermicro.com/news/news-details/2026/Supermicro-Delivers-NVIDIA-Vera-Rubin-NVL4-End-to-End-DCBBS-Blueprint-with-Native-FP64-Performance-for-Converged-HPC-and-AI-Infrastructure/default.aspx

2. ⭐ **NVIDIA Vera Rubin 已进入全量产爬坡叙事**
   - 要点：NVIDIA 官方披露 Vera Rubin 供应链开始规模制造，并把 NVL72、BlueField-4 storage 与 Spectrum-6 Ethernet rack 组合为系统平台。
   - 技术视角：Rubin 的性能兑现依赖 NVLink scale-up、Ethernet/InfiniBand scale-out、storage fabric 与液冷的协同，而不是单颗 GPU 指标。
   - 来源：https://investor.nvidia.com/news/press-release-details/2026/NVIDIA-Vera-Rubin-Ramps-Into-Full-Production-to-Power-Agentic-AI-Factories-Worldwide/default.aspx

3. ⭐ **Vera CPU 与 Rubin GPU 采用 NVLink-C2C 构成 coherent compute node**
   - 要点：NVIDIA 官方给出的 CPU-GPU coherent bandwidth 为 1.8TB/s，并强调 Vera 的 memory subsystem 与 Scalable Coherency Fabric。
   - 技术视角：CPU-GPU coherent domain 扩大后，scale-up fabric 的 memory ordering、RAS 和 congestion 行为更接近系统架构问题。
   - 来源：https://investor.nvidia.com/news/press-release-details/2026/NVIDIA-Launches-Vera-CPU-Purpose-Built-for-Agentic-AI/default.aspx

## 🏢 计算中心与基础设施

1. 🔥 **FERC 要求六个区域电网运营方解释或改革大负载并网规则**
   - 要点：6 月 18 日行动直接针对 data center、制造设施等 large loads 的 grid connection 流程。
   - 技术视角：并网排队和供电可得性正在成为 AI cluster 上线周期的一级约束，推动 load flexibility、HVDC 与分阶段扩容。
   - 来源：https://www.ferc.gov/news-events/news/ferc-launches-aggressive-targeted-action-speed-large-load-integration

2. ⭐ **Schneider Electric 与 Foxconn 联合定义 AI data center 的电力、冷却与数字化栈**
   - 要点：双方 6 月 15 日宣布合作，将 integrated power、cooling 与 digital capabilities 纳入 AI factory 规模化方案。
   - 技术视角：服务器 ODM 与设施供应商的边界正在融合，rack power/cooling interface 会更早进入平台定义。
   - 来源：https://www.nasdaq.com/press-release/schneider-electric-and-hon-hai-technology-group-foxconn-announce-strategic-0

3. ⭐ **新论文系统梳理 48V、LVDC 与 medium-voltage power conversion 的迁移路径**
   - 要点：arXiv:2606.25095 将高变比 DC/DC、facility-level LVDC 和 medium-voltage solid-state transformer 列为下一代 AI data center 的三类关键 building blocks。
   - 技术视角：当 rack power density 与 load transient 同时上升，供电架构会像网络拓扑一样成为计算平台的设计变量。
   - 来源：https://arxiv.org/abs/2606.25095
