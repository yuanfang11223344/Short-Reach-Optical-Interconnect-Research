# 芯片互连每日资讯 2026-06-29

> 生成时间：2026-06-29（Asia/Shanghai）。周一日报先行生成，并作为 2026-W27 周报的首日素材；公开新增不足的方向以一手状态复核补齐。

## 🔌 SerDes与互连技术

1. 🔥 **P802.3dj Draft 3.1 把 224G SerDes 推入跨层一致性验证阶段**
   - SA ballot comments 已公开，200G/lane electrical/optical PMD、FEC 与管理条款进入收敛。
   - 技术视角：后续风险主要来自 reference package、channel model、test fixture 与 DSP/FEC 假设不一致，而不只是眼图余量。
   - 来源：https://www.ieee802.org/3/dj/comments/index.html

2. 🔥 **OIF 已把 448G 从 framework 推进到 VSR/LR 项目**
   - OIF 的 OFC 2026 材料显示 CEI-448G-VSR 与 CEI-448G-LR 已项目化，应用边界覆盖 chip-to-chip、backplane/cable 与 chassis 间连接。
   - 技术视角：448G PAM4 将迫使 ADC/DAC、clocking、equalization、package 和测试能力同时升级。
   - 来源：https://www.oiforum.com/technical-work/current-work/

3. ⭐ **224G retimer 开始承担多协议 telemetry 与 RAS**
   - Credo Blue Heron 同时支持 UALink、ESUN 与 Ethernet，显示 PHY 平台复用将推动 latency、BER telemetry、lane repair 与 compliance 的统一实现。
   - 来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Industrys-First-224G-Multiprotocol-AI-Scale-Up-Retimer-Supporting-UALink-ESUN-and-Ethernet/default.aspx

## 🔦 光学与光模块

1. 🔥 **Spectrum-X Ethernet Photonics 将 CPO 纳入量产 AI 网络平台**
   - NVIDIA 官方称该平台已进入 production，并与 Vera Rubin、Spectrum-6 Ethernet racks 共同构成 million-GPU AI factory 网络路线。
   - 技术视角：CPO 商业化从独立 demo 走向交换芯片、网络软件、运维和供应链协同，但可维护性与 ELS 仍是关键工程点。
   - 来源：https://investor.nvidia.com/news/press-release-details/2026/NVIDIA-Vera-Rubin-Ramps-Into-Full-Production-to-Power-Agentic-AI-Factories-Worldwide/default.aspx

2. 🔥 **OpenLight 3.2T DR8 PIC 验证 400G/lane EAM 路线**
   - 官方提供 3.2T DR8 silicon photonics PIC 样片，并同步推出 1.6T DR8 LRO/LPO variants。
   - 来源：https://www.ofcconference.org/news-media/exhibitor-news/openlight-introduces-3-2t-dr8-silicon-photonics-pic-s-as-well-as-1-6t-dr8-lro-and-lpo-variants/

3. ⭐ **Socketed CPO、NPO 与 OBO 将继续并行**
   - Coherent 的 socketed CPO 与 Lightmatter L20 的 NPO/OBO 兼容路线，都在降低 fully co-packaged 架构的维护和制造风险。
   - 来源：https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026
   - 补充：https://www.ofcconference.org/news-media/exhibitor-news/lightmatter-expands-photonic-interconnect-roadmap-with-passage-l20-unified-optical-engine/

4. ⭐ **国内制造端开始围绕 800G/1.6T/3.2T 建立全流程工艺展示**
   - NEPCON ASIA 的示范线覆盖固晶、键合、耦合、SMT、点胶、测试老化，说明竞争焦点正从样机扩展到良率、自动化与测试吞吐。
   - 来源：https://www.nepconasia.com/zh-cn/news-center/zhxx/2026/6/1.html

## 🖥️ AI服务器与超节点

1. 🔥 **Vera Rubin 以 NVL72 + storage + Ethernet rack 形成 POD-scale 架构**
   - NVIDIA 将 Vera CPU、Rubin GPU、NVLink 6、BlueField-4 与 Spectrum-6 作为统一系统交付。
   - 技术视角：scale-up memory semantics、scale-out congestion control、storage checkpoint 和 liquid cooling 必须共同验证。
   - 来源：https://developer.nvidia.com/blog/inside-the-nvidia-rubin-platform-six-new-chips-one-ai-supercomputer/

2. ⭐ **Supermicro DCBBS 将服务器交付边界上移到 facility**
   - Vera Rubin NVL4 blueprint 同时覆盖 compute、networking、advanced liquid cooling、power distribution 与 site infrastructure。
   - 来源：https://ir.supermicro.com/news/news-details/2026/Supermicro-Delivers-NVIDIA-Vera-Rubin-NVL4-End-to-End-DCBBS-Blueprint-with-Native-FP64-Performance-for-Converged-HPC-and-AI-Infrastructure/default.aspx

3. ⭐ **UALink/ESUN/Ethernet 的 PHY 共平台趋势更加明确**
   - 224G multiprotocol retimer 的出现意味着 scale-up fabric 差异将更多上移到 link layer、memory semantics、congestion control 和 software orchestration。
   - 来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Industrys-First-224G-Multiprotocol-AI-Scale-Up-Retimer-Supporting-UALink-ESUN-and-Ethernet/default.aspx
