# 芯片互连每日资讯 2026-07-06

> 检索窗口：截至 2026-07-06 17:00（Asia/Shanghai）。今天是周一，本日报先生成并作为 `2026-W28` 周报的首日素材。

## 🔌 SerDes与互连技术

1. 🔥 **OIF CEI-448G-VSR/LR 进入具体接口项目。** VSR 面向 chip-to-module，LR 面向 backplane；448G 的核心工作从“是否可行”转为 channel、package、modulation/DSP power 和 compliance 定义。[OIF](https://www.oiforum.com/technical-work/current-work/)
2. 🔥 **P802.3dj 进入 7月 plenary 前的 D3.x 收敛窗口。** 近期 optics ad hoc 聚焦 TDECQ、reference equalizer 与 COM runtime，说明 200G/lane 正转向一致性测试模型。[IEEE](https://ieee802.org/3/dj/public/adhoc/optics/index.html)
3. 🔥 **Broadcom Taurus 把 400G/lane DSP 引入产品周期。** 8:4 optical DSP 连接当代 1.6T 与下一代 3.2T module，关键挑战是 ADC/DAC、laser driver、FEC 与功耗共同收敛。[Broadcom](https://investors.broadcom.com/news-releases/news-release-details/broadcom-delivers-industrys-first-400glane-optical-dsp-next)
4. ⭐ **Synopsys 公开支持 UALink 2.0，并强调 224G 与 security IP。** 这表明开放 scale-up fabric 的实现栈正在从规范扩展到可授权 PHY、controller、security 与 verification IP。[UALink](https://ualinkconsortium.org/news/statements-of-support-for-the-ualink-2-0-specification/)

## 🔦 光学与光模块

1. 🔥 **Spectrum-X Ethernet Photonics 将 CPO 纳入 Rubin 网络平台。** NVIDIA 官方称 CPO switch systems 面向更高 power efficiency 和 uptime，验证边界包括 switch silicon、optical engine、network OS 与运维。[NVIDIA](https://nvidianews.nvidia.com/news/rubin-platform-ai-supercomputer)
2. 🔥 **3.2T 正形成 heterogeneous SiPh、SOH 与 InP 多路线竞争。** NLM 的 SOH PIC sampling、Coherent 的 400G InP-on-silicon modulator 与 Broadcom 400G/lane DSP 分别覆盖调制、器件和 DSP 关键环节。[NLM](https://www.ofcconference.org/news-media/exhibitor-news/nlm-photonics-initiates-sampling-of-1-6t-and-3-2t-silicon-organic-hybrid-pics/)
3. ⭐ **Coherent 6.4T socketed CPO 继续强调可维护性折中。** socketed engine 与 ELS 尝试降低 fully integrated CPO 在 yield、laser replacement 和 field service 上的风险。[Coherent](https://www.coherent.com/news/press-releases/coherent-co-packaged-optics-cpo-technologies-ofc-2026)
4. ⭐ **国内厂商把产品路线延伸到 6.4T NPO、12.8T XPO 与 OCS。** 中际旭创董事会报告将 3.2T、1.6T Coherent Lite、6.4T NPO、12.8T XPO 和 OCS 列入研发方向；这是路线规划，不等同于量产承诺。[巨潮资讯](https://static.cninfo.com.cn/finalpage/2026-03-31/1225056493.PDF)

## 🖥️ AI服务器与超节点

1. 🔥 **UALink Common 2.0 已将 In-Network Compute 纳入可用规范。** reductions、aggregations、synchronization 和 collectives 可在 fabric 内执行，降低 endpoint 数据往返。[UALink](https://ualinkconsortium.org/specification/)
2. ⭐ **UALink Management 与 Chiplet specifications 扩大开放生态边界。** gNMI/Redfish manageability 和 chiplet integration 让开放 scale-up fabric 更接近可部署、可运维的平台。[UALink 支持声明](https://ualinkconsortium.org/news/statements-of-support-for-the-ualink-2-0-specification/)
3. 🔥 **Rubin 以六芯片 extreme codesign 绑定 scale-up、scale-out 与 storage。** NVLink 6、ConnectX-9、BlueField-4 和 Spectrum-6 与 CPU/GPU 同时定义 release gate。[NVIDIA](https://developer.nvidia.com/blog/inside-the-nvidia-rubin-platform-six-new-chips-one-ai-supercomputer/)

## 🏢 计算中心与基础设施

1. 🔥 **AI data center 的供电架构从 rack 级 48V 上移到 facility-level DC。** 高变比 DC/DC、LVDC 和 medium-voltage SST 分别解决转换级数、铜损和中压接入问题。[arXiv:2606.25095](https://arxiv.org/abs/2606.25095)
2. ⭐ **液冷系统从冷板向 CDU/chiller 全链路扩展。** Panasonic 的 400/800kW CDU 和 800/1200kW chiller 产品说明设施 loop、冗余和控制接口成为交付重点。[Panasonic](https://news.panasonic.com/global/press/en260304-2)
3. ⭐ **AI load 与电网协同成为基础设施设计变量。** 欧盟相关项目和电力系统论文都把负载灵活性、可持续性与 grid operation 联合研究。[European Commission](https://energy.ec.europa.eu/news/flagship-projects-ai-grids-and-data-centre-sustainability-2026-06-04_en)
