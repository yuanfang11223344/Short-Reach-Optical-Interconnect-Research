# 芯片互连每日资讯 2026-07-14

## 🔌 SerDes与互连技术

- 🔥 OIF OFC 2026 CEI rolling deck 把 224G VSR multi-vendor interop、212.5Gbps PRBS31Q PAM4 和 1.6T retimed connectivity 的热设计放在同一组 demo 中；224G 量产风险已经从单颗 PHY 扩展到模块板、散热器和系统互通。来源：https://www.oiforum.com/wp-content/uploads/CEI-Rolling-Deck.pdf
- 🔥 OIF CEI-448G framework 已发布，明确 448Gbps/lane 需要新的 specification 与 technology，用于下一代通信、计算和交换硬件互连；这会牵动 3.2T/6.4T optical module host interface 规划。来源：https://www.oiforum.com/wp-content/uploads/OIF-FD-CEI-448G-01.0.pdf
- ⭐ Marvell 224G LR SerDes 文章给出 2.5m copper channel、1e-11 BER 的 scale-up/scale-inside 叙事，说明 224G copper reach 仍可服务 rack 内 flexible placement。来源：https://www.marvell.com/blogs/224g-long-range-serdes-scale-up-scale-inside.html
- ⭐ Synopsys 224G Ethernet PHY IP 页面强调面向高性能 data center 的 bandwidth/latency 需求，并列出 DesignCon 2026 互通 demo；可作为 IP 选型时观察 interop maturity 的入口。来源：https://www.synopsys.com/designware-ip/interface-ip/ethernet/224g-ethernet-phy.html

- ⭐ Credo Cardinal 1.6T optical DSP family 支持 retimed 1.6T optics 与 linear-receive implementations，反映 1.6T module 正在围绕功耗、延迟和是否保留完整 DSP 重新权衡。来源：https://investors.credosemi.com/news-events/news/news-details/2026/Credo-Introduces-Cardinal-A-LowPower-1-6T-Optical-DSP-Family-Engineered-for-MassiveScale-AI-Fabrics/default.aspx

- 📌 光迅科技披露 1.6T 光模块具备批量交付能力，且 OFC 2026 推出 3.2T 硅光单模 NPO；国内光模块厂商正把硅光和 NPO 作为差异化路线。来源：https://vip.stock.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=12321776&stockid=002281

## 🖥️ AI服务器与超节点

- 🔥 NVIDIA NVLink 官方页继续突出 NVLink Switch 在 rack-scale GPU-to-GPU 通信中的地位，NVLink Fusion 把第三方 CPU/XPU 接入也纳入 fabric；对网络工程师而言，scale-up fabric 已从板级互连演进为系统级产品边界。来源：https://www.nvidia.com/en-us/data-center/nvlink/
- 🔥 UALink 白皮书将 UALink 定位为开放、高效率 scale-up interconnect，连接 PCIe/CXL node-local 与 Ethernet/InfiniBand scale-out 之间的空白；其价值在多厂商 XPU rack-scale 生态。来源：https://ualinkconsortium.org/wp-content/uploads/2026/01/UALink_White_Paper_Publication_Candidate_FINAL_VERSION.pdf
- ⭐ Synopsys 文章把 UEC 与 UALink 分别对应 scale-out 与 scale-up，并强调二者依赖 224G SerDes PHY；协议开放化不会降低 PHY 难度，反而提高互通验证权重。来源：https://www.synopsys.com/articles/224g-serdes-interoperability-ai-hpc.html
- 📌 Astera Labs UALink 视频说明其围绕 UALink 产品组合支持 rack-scale AI connectivity，Retimer/Smart Cable/PCIe-CXL 生态供应商会成为开放 scale-up 的早期受益者。来源：https://www.asteralabs.com/videos/rack-scale-ai-connectivity-with-ualink/
